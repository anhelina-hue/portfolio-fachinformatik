## Projekte

### Quaderberechnung 
Das Programm berechnet anhand der drei Seitenlängen eines Quaders:
- das Volumen
- die Oberfläche
- die Raumdiagonale

### Benzinverbrauchsrechner
Diese Anwendung berechnet den Benzinverbrauch eines Fahrzeugs. Der Benutzer gibt dabei den Verbrauch in Litern und die gefahrenen Kilometer ein,
wenn ein Button gedrückt wird, zeigt das Programm den Verbrauch in Litern pro 100 km an. Zusätzlich wird der in  den USA gebräuchliche Wert "Miles per gallon"
berechnet und angezeigt.

### Mini-Spiel: Zahlenraten
Dieses Konsolenprogramm in C# generiert eine Zahl zwischen 1 und 10. 
Nachdem der Benutzer geraten hat, gibt das Programm zurück, ob die Zahl 
zu groß, zu klein oder richtig war. 

Am Ende gibt es die Geheimzahl aus. 
Das Programm wird mit einem Tastendruck beendet.

#### Verwendete Konzepte:
- if/else if/else Bedingungen
- Vergleichsoperatoren
- Random:
```csharp
Random random = new Random();
int geheimzahl = random.Next(1, 11);
```
- Schleifen
- Konsolenein- und -ausgabe
